# 🚀 Sistema Inteligente de Rutas

## 📌 Descripción

Este proyecto consiste en el desarrollo de un sistema inteligente basado en reglas que permite encontrar la mejor ruta entre dos puntos dentro de un sistema de transporte.

El sistema utiliza una representación del conocimiento mediante grafos, donde cada nodo representa una estación y cada conexión tiene un costo en tiempo y distancia.

Para la optimización de rutas se implementa el algoritmo de Dijkstra, el cual permite encontrar el camino más eficiente en términos de tiempo.

---

## 🧠 Características

- Representación del sistema mediante grafos
- Uso de reglas para la toma de decisiones
- Implementación del algoritmo de Dijkstra
- Cálculo de tiempo y distancia total
- Comparación con una ruta alternativa no optimizada

---

## ⚙️ Tecnologías utilizadas

- Python 3

---

## ▶️ Ejecución del programa

1. Clonar el repositorio:

~~~bash
git clone https://github.com/TU-USUARIO/TU-REPO.git
~~~

2. Ejecutar el archivo:

~~~bash
python rutas_IA.py
~~~

3. Ingresar el punto de inicio y destino cuando el sistema lo solicite.

---

## 🧪 Ejemplo de uso

~~~
Inicio: F
Destino: A

Ruta óptima:
F → E → C → A
Tiempo total: 7 minutos
Distancia total: 14 km
~~~

---

## 📊 Funcionamiento

El sistema evalúa diferentes rutas posibles utilizando el algoritmo de Dijkstra, seleccionando aquella con menor costo en tiempo.

Adicionalmente, se genera una ruta alternativa basada en selección aleatoria, lo que permite comparar el rendimiento del sistema inteligente frente a decisiones no optimizadas.
